# Udacity-Travel-App
Simple application that gets destination and trip date from user, and then returns an image of the location alongside the weather forecast  

## Description
Simple application that takes in the location, departure and return dates from the user and then returns an image of the location along with the weather forecast
It uses three APIs to achieve this: 
- Geonames
- Weatherbit
- Pixaby

### Dependencies
- Body parser, cors, express, mini-css-extract-plugin, webpack, webpack-cli
- Babel/core, babel/preset-env, babel-loader, style-loader, css-loader, terser-webpack-plugin, node-sass, sass-loader, webpack-dev-server
